# Connect 4 AI Agent

## Overview

This project implements an intelligent Connect 4 Game Playing Agent using Artificial Intelligence and Machine Learning techniques. The agent follows a Think-Act Cycle where it perceives the current board state, analyses possible moves, and selects an optimal action based on its strategy.

The system includes three AI approaches:

- **Smart Agent** – Uses rule-based decision making to identify winning moves, block opponents, and select valid moves.
- **Mini-Max Agent** – Uses adversarial search with heuristic evaluation and Alpha-Beta pruning to determine the best possible move.
- **Machine Learning Agent** – Learns from Connect 4 game data and predicts moves based on previously observed board states and outcomes.

The game supports:
- Human vs AI gameplay
- Mini-Max Agent vs ML Agent gameplay
- Automated AI performance testing over multiple games

Performance is evaluated using metrics such as:
- Win rate, loss rate, and draw rate
- Search depth and nodes expanded
- Execution time
- Alpha-Beta pruning efficiency
- Game strategy effectiveness
